# Rock vs. Mine Predictor

Binary classifier that distinguishes rocks from underwater mines 
using sonar frequency data.

## Tech Stack
Python, pandas, scikit-learn, Logistic Regression

## Results
- 83% training accuracy
- 76% test accuracy

## Dataset
UCI Sonar Dataset — 208 samples, 60 frequency features

## How to Run
1. Clone the repo
2. pip install pandas scikit-learn numpy
3. Open RockvsMinePrediction.ipynb in Jupyter
